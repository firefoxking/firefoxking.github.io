Centos7 添加自己开机自启动脚本的方法

可以有很多种添加开机自启脚本的方法，我这里写一个自己喜欢的。

  1. 自己新建一个需要启动自运行的脚本

    autostart.sh

    1
2

    #!/bin/bash
echo start >> /home/user-for-dockerStudy/linux/test.txt

  2. 赋予脚本可执行权限

    1

    $ chmod +x /home/user-for-dockerStudy/linux/autostart.sh

  3. 打开/etc/rc.d/rc.local文件,在最后面添加如下脚本

    1

    /home/user-for-dockerStudy/linux/autostart.sh

  4. 在centos7中，/etc/rc.d/rc.local的权限被降低了，所以需要执行如下命令赋予其可执行权限

    1

    $ chmod +x /etc/rc.d/rc.local
