4. 图的拓扑排序

拓扑排序：对于有向无环图G = (V, E) 而言，其拓扑排序就是G 中所有节点的一种线性次序， 使得图中任意一对顶点u和v，若边 (u,v)∈E(G) ，则u在线性序列中出现在v之前。对于一个图来说，拓扑排序可能不唯一。

拓扑排序算法过程：

  1. 选择一个入度为0的顶点并输出。
  2. 从图中删除该顶点及其所有出边。
  3. 循环执行以上两步，直到不存在入度为0的顶点为止；如果剩下入度非0的顶点（输出的顶点数小于图中的顶点数），就说明有回路，不存在拓扑排序。

示例：

将上图用字典表示：G = { ‘a’:‘bf’, ‘b’:‘cdf’,‘c’:’d',’d':‘ef’,‘e’:‘f’,f:''}

def topSort(graph):
    # 初始化入度为0
    in_degrees = dict((u, 0) for u in graph)
    num = len(in_degrees)
    # 计算每个顶点的入度
    for u in graph:
        for v in graph[u]:
            in_degrees[v] += 1
    Q = [u for u in in_degrees if in_degrees[u] == 0]
    Seq = []
    while Q:
        u =  Q.pop()
        Seq.append(u)
        for v in graph[u]:
            in_degrees[v] -= 1
            if in_degrees[v] == 0:
                Q.append(v)
    if len(Seq) == num:
        return Seq
    else:
        return None


G = {
    'a':'bf',
    'b':'cdf',
    'c':'d',
    'd':'ef',
    'e':'f',
    'f':''
}
print(topSort(G))

>>> ['a', 'b', 'c', 'd', 'e', 'f']

5. 最小生成树算法

解决最小生成树问题常用的有 Prim算法 和 Kruskal算法 ，二者均基于贪心算法。 Prim算法 思想很简单，即每步都沿着最小权重的边向前搜索，找到一条权重最小的可以安全添加的边，将边添加到树中。 Kruskal算法 是对所有的权重进行排序，再次体现贪心算法的思想。通过对权重排序，将权重最小的边开始遍历，直至图中的所有节点都连在同一个树上，此时即为最小生成树。

Prim 算法

prim算法过程：

  1. 指定根节点r，划分两个集合，A 存已经在生成树中的节点、B 存还不在生成树中的节点

  2. 找到权值最小的边（边的节点一个在 A 中，一个在 B 中）

  3. 循环知道所有的节点都在A中。

    def prim(self):
        res = []
        if self.nodenum <= 0 or self.edgenum < self.nodenum-1:
            return res
        res = []
        seleted_node = [0]
        candidate_node = [i for i in range(1, self.nodenum)]

        while len(candidate_node) > 0:
            begin, end, minweight = 0, 0, 9999
            for i in seleted_node:
                for j in candidate_node:
                    if self.maps[i][j] < minweight:
                        minweight = self.maps[i][j]
                        begin = i
                        end = j
            res.append([begin, end, minweight])
            seleted_node.append(end)
            candidate_node.remove(end)
        return res

    这里找权值最小的边的过程是采用遍历的手段，速度较慢。可以使用最小优先队列数据结构存储到A集合中，A集合的节点v包含两个属性，一个是连接v与树中节点的所有边中最小边的权重，另一个则是节点v在树中的父节点。

kruskal 算法

kruskal 算法过程：

  1. 初始化一个空集，存储生成树的边，并创建节点数目的N个树，每棵树仅包含一个节点

  2. 对图的所有边按权值从小到大顺序排序

  3. 按从小到大的次序对边进行检查，如果同属一棵树跳过该边，否则增加这条边，并对树进行合并

    def kruskal(self):
        res = []
        if self.nodenum <= 0 or self.edgenum < self.nodenum-1:
            return res
        edge_list = []
        for i in range(self.nodenum):
            for j in range(i,self.nodenum):
                if self.maps[i][j] < 9999:
                    edge_list.append([i, j, self.maps[i][j]])#按[begin, end, weight]形式加入
        edge_list.sort(key=lambda a:a[2])#已经排好序的边集合

        group = [[i] for i in range(self.nodenum)]
        for edge in edge_list:
            for i in range(len(group)):
                if edge[0] in group[i]:
                    m = i
                if edge[1] in group[i]:
                    n = i
            if m != n:
                res.append(edge)
                group[m] = group[m] + group[n]
                group[n] = []
        return res
