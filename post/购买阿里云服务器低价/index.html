一、购买云服务器 ECS

学生党可以购买便宜的阿里云服务器，用于实现自己的个人博客。

ECS (Elastic Compute Service)

  1. 登录阿里云并实名认证：https://developer.aliyun.com/plan/student。按照它的步骤进行就可以获得一台服务器。

  2. 远程连接服务器

    • 重置服务器密码并重启

    • 使用远程工具连接 （finalshell）

    OK， 已经连接上了。

  3. 简单管理服务器

    使用宝塔Linux面板管理服务器。https://www.bt.cn/download/linux.html

    • 阿里云 ECS 放行8888端口及其他端口

    • 运行centos系统对应宝塔安装配置命令

      1

      yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

      ==虽然我开放了8888端口，但是还是不能访问宝塔控制==

    只要安装了宝塔控制就可以很简单安装环境。
