236. 二叉树的最近公共祖先

从左树要信息，从右树要信息，然后归总信息。是典型的后序遍历方式。

这道题需要的子树信息是子树上是否含有待求节点p和q，如果有一个就返回子树的根，如果同时包含p和q则是返回最近祖先，如果没有就返回null。信息归总时，就是观察子树返回的结果，左右子树都为空则此时的树也不包含返回null、一个子树返回空一个不为空则此时的树可能为包含p、q或只包含一个，但返回值都是也一样的、两个子树都返回非空，表示当前树根节点就是最近祖先。

这样的子树返回设置实则丢弃了部分信息，就是他可能产生信息歧义，但是由于题目特定语义使得这种设置是正确的。但如果题目变为：

  1. p、q节点在整棵树上都有可能不存在。

  2. p、q节点”存在多次“，这里我的意思是如果给的p、q是节点的val，而树中所有的val可能重复

    这些情况上面就不对，需要把子树返回信息写完整。

class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root == null)
            return null;
        if (root == p || root == q)
            return root;
        TreeNode left = lowestCommonAncestor(root.left, p, q);
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        if (left != null && right != null)
            return root;
        return left != null ? left : right;
    }
}

class Solution {
    public String addStrings(String num1, String num2) {
        StringBuilder sb = new StringBuilder();
        char[] str1 = num1.toCharArray();
        char[] str2 = num2.toCharArray();
        int i1 = str1.length - 1;
        int i2 = str2.length - 1;
        int c = 0;
        while (c != 0 || i1 >= 0 || i2 >= 0) {
            int a = i1 >= 0 ? str1[i1] - '0' : 0;
            int b = i2 >= 0 ? str2[i2] - '0' : 0;
            int temp = a + b + c;
            sb.append(temp % 10);
            c = temp / 10;
            i1--;
            i2--;
        }
        return sb.reverse().toString();
    }
}

class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode dummy = new ListNode();
        dummy.next = head;
        ListNode p = dummy;
        ListNode newHead = p;
        while (p != null) {
            // 走k步,判断是否足够
            for (int i = 0; i < k && p != null; i++) {
                p = p.next;
            }
            if (p == null)
                break;
            // 头插法反转k个节点
            ListNode cur = newHead.next;
            for (int i = 0 ; i < k - 1; i++) {
                ListNode nextNode = cur.next;
                cur.next = nextNode.next;
                nextNode.next = newHead.next;
                newHead.next = nextNode;
            }
            // 重新设置头插法的头节点
            newHead = cur;
            p = newHead;
        }
        return dummy.next;
    }
}

33. 搜索旋转排序数组

核心：对于一个旋转排序数组来说，以任意一点分割数组为两部分，必有一个部分是有序的。

class Solution {
    public int search(int[] nums, int target) {
        int l = 0, r = nums.length - 1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] == target)
                return mid;
            else {
                if (nums[l] <= nums[mid]) {
                    if (target >= nums[l] && target < nums[mid]) {
                        r = mid - 1;
                    } else {
                        l = mid + 1;
                    }
                }
                if (nums[mid] <= nums[r]) {
                    if (target > nums[mid] && target <= nums[r])
                        l = mid + 1;
                    else
                        r = mid - 1;
                }
            }
        }
        return -1;
    }
}
